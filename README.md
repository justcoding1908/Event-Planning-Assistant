# Event Planning Assistant with Budget Tracker

## Project Overview
The Event Planning Assistant with Budget Tracker is an AI-based application designed to simplify the process of planning events such as weddings, parties, and corporate gatherings. The system will assist users by providing intelligent suggestions for event planning and by helping them manage and track their event budget effectively.

The project focuses on combining Large Language Models (LLMs) with data visualization and cloud technologies to build an interactive and user-friendly solution.

## Objectives
- To assist users in planning events through AI-generated recommendations
- To provide structured timelines and planning guidance
- To track event-related expenses in a clear and organized manner
- To visualize budget data for better financial decision-making
- To support multi-user access through cloud deployment

## Technologies Planned
- Large Language Models (LLM) for chatbot-based recommendations
- Data Visualization Tools for budget and expense analysis
- Cloud Platform for hosting and multi-user access
- GitHub for version control and collaboration

## Project Workflow 
1. Requirement analysis and project planning
2. Designing chatbot interaction flow
3. Designing budget tracking and visualization module
4. Integrating components into a single system
5. Cloud deployment and testing
